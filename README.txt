Agar Plate Image Processing for Jupyter Lab Notebooks, ChE126 FA 2019

Description
- Images for malt extract agar (MEA) and MEA + Chloramphenicol (MEACHL) plates innoculated with soil dilutions (10^-1 - 10^-5) from rotting Quercus agrifolia. 
- csv files of Absorbance over time for fungal growth
- Bespoke Python module,CFUcounter.py, I developed for automated and manual CFU counting and plate coverage estimations. See github.com/theo-yang for updates.
- 

 
Naming Conventions
- MEA: Malt extract agar only
- MEACHL: Malt extract agar and 100 mg/mL Chlorampenicol
- Images are named "treatment_environment_-log10(dilution).JPG"
